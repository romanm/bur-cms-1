Створити BuhTinyWiki -- Дуже маленький WIKI

Динамічний сайт по принципу WIKI -- редагуеш і одразу бачиш сторінку, за найпримітивнішим принципом.

Постановка задачі

  1. Один json файл -- багато сторінок сайту
  2. Вибір певної сторінки з json файла через ?uniquePageName підхід і REST підхід.
  3. Два html файла -- один для перегляду, другий для редагування
  4. Редагування з паролем -- перегляд вільний
  5. Структута json, що дозволяє реалізувати попередні пункти.

Реалізація

Базісна реалізація

  1. Підключення W3C bootstrap company
  2. Зчитування даних з property файлів
  3. Site map editor
  4. BuhTinyWiki view
  5. Редагування сторінки
