Перший крок створення buh-cms1 -- Буг система керування контентом

Постановка задачі

  1. Створити систему керування контентом силами студентів університетів річки Південий Буг (buh-cms1).
  2. Власником buh-cms1 є і залишається спільнота студентів розробників.
  3. Мета buh-cms1 надати можливість студентам вдосконалювати себе на теренах опанування майбутньої професії.
  4. buh-cms1 доступна з вільним кодом з вказанням прав власності.
  5. buh-cms1 базується на сучасних java web технологіях.
  6. buh-cms1 призвана бути базісом для написання і захисту дипломних робіт в сфері IT технології.
